##  Overview
The **Invoice ** is a program that produces professional invoices for multiple products or services.  
It ensures proper alignment, formatting, and compliance with assignment rules (e.g., no arrays, no loops if required).  
The output is designed to resemble a real business invoice, making it suitable for academic submissions or real-world use.

##  Features
- Input customer details such as name.
- Add multiple products with:
  - Product name
  - Quantity
  - Unit price
  - Total per product
- Automatic calculation of:
  -  Subtotal  
  -  Taxes (if applicable)  
  -  Grand Total
- Pixel-perfect alignment and professional formatting.
- Console-based output that matches sample invoice images.
